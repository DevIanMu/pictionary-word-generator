# Pictionary Word Generator

A free, fun, and fast Pictionary word generator built for game nights. Features 500+ words across 11 categories with 3 difficulty levels, plus a built-in timer.

## Features

- 🎲 **500+ Words** across 11 categories (Animals, Food, Actions, Places, Nature, Sports, Professions, Emotions, Movies, Idioms, Objects)
- ⭐ **3 Difficulty Levels** — Easy, Medium, Hard
- ⏱️ **Built-in Timer** with adjustable duration (30s/60s/90s/120s)
- 🎨 **Party-themed Design** with card-flip animations and confetti
- 📱 **Fully Responsive** — works great on mobile and desktop
- 🔍 **SEO Optimized** — structured data, meta tags, fast loading

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Vite (build tool)
- Google Fonts (Baloo 2 + Inter)
- No external JS dependencies

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output will be in `dist/` directory.

## Deploy

Deploy the `dist/` folder to any static hosting (Vercel, Netlify, Cloudflare Pages, etc.).
